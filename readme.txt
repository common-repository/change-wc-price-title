This easy-to-use plugin allows WooCommerce store owners to change the WooCommerce Price Title. Also, it is possible to hide the price and option to apply the settings on individual or all WooCommerce pages.

== Description ==

This plugin allows you to change the WooCommerce Price Title. For e.g From: $100.

You can completely change price title or set caption to price using PRICE shortcode.

Also, they have ability to change/hide price title on individual WooCommerce Product page OR on all WooCommerce pages.

== Installation ==

Method 1:
<ol>
<li> Download the plugin change-wc-price-title.zip </li>
<li> Unpack the change-wc-price-title.zip file and extract the change-wc-price-title folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the Change Price Title for WooCommerce and click on Activate link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin change-wc-price-title.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded change-wc-price-title.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>
